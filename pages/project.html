Ultrasonic Sensing of Objects and DE2Bot Programming

I have led a 4-person team to complete a class project in Digital Design Lab. The project required that we locate and tag objects using ultrasonic range-finders installed on a DE2Bot. The DE2Bot is comprised of the commercially-available AmigoBot with its electronics removed and replaced with custom hardware and an Altera DE2 FPGA development board. The configuration allows compete control of the robot hardware using custom digital circuits created within the FPGA. Our team took an approach that consisted of three main routines implemented in assembly code:

  1. A routine to determine the path the robot will travel along in search of objects
  2. A routine to detect objects around the robot
  3. A routine to tag the objects

We developed path and sensing algorithms by using ultrasonic sensors to estimate the location of objects and adjusting steering to stay within the desirable area. Our design received fifth place out of approximately forty groups during the final demonstration. Our team documented our design into a design proposal, a design report, a demonstration of the capabilities, and a PowerPoint presentation that we deliver our algorithms to other groups in the lab.


Mbed version of Missile Command Video Arcade Game

I created a missile command video arcade game using Mbed microcontroller. The game was implemented in C. The general algorithm used was a doubly linked list. Game features include aircraft movement, the ability to fire missiles, detecting missile interception points, animating explosions, detecting and animating city destruction, and displaying the game score. The level also advances when a certain amount of missiles has been intercepted or the user can force level advance using the push button. The code was documented to allow better understanding of the code for others.
